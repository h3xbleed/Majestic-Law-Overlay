Глава 2. Доступ к государственной волне

2Интервал вещания в госволну
Каждый сотрудник государственной организации имеющий доступ к аппаратуре вещания в обязательном порядке должен соблюдать прописанный в данном законе интервал:1.Занимать государственную волну разрешено не менее чем за 10 минут и не более чем за 60 минут;2.Интервал времени между каждым вещанием в государственную волну должен быть не менее 20 минут для одной организации, 10 минут - общий интервал;3.Время вещания должно быть в определенное время, десятые минуты по местному времени;Пояснение: К десятым минутам относятся, к примеру, 0:10; 0:20; 0:30; 0:40; 0:50.Исключение: Экстренные новости о чрезвычайном и военном положении.
3Темы для вещания в госволну
Темы, которые могут быть упомянуты на момент вещания в государственную волну:1.Начало собеседования, призыва или рекрутинга в государственную организацию;2.Начало регистрации на мероприятия от государственной организации;3.Напоминание о проходящем собеседовании, призыве или рекрутинге в государственную организацию и о мероприятиях;4.Конец проходящего собеседования, призыва или рекрутинга в государственную организацию, конец регистрации на мероприятия;5.Информирование о прохождении открытого судебного заседания;6.Информирование населения о происходящей чрезвычайно ситуации, военном положении;7.Технические неполадки;8.Оповещение про продажу, продление лицензий;
6Освобождение госволны после вещания
После окончания вещания в государственную волну сотрудник государственной структуры имеющий доступ к аппаратуре вещания обязан сообщить в общую рацию департамента о том, что тот освобождает государственную волну.

Глава 4. Волны государственного назначения

1Волны государственного назначения
Волнами государственного назначения признаются все общие волны государственных организаций, которые используются следующими организациями: LSPD, LSCSD, FIB, GOV, EMS, SANG, Weazel News. Также волнами государственного назначения признается диапазон частот 100.000 — 100.699, зарезервированный под использование теми или иными государственными организациями. Также волной государственного назначения признается общая волна департамента.
3.1Закреплённые частоты госструктур
Свод закрепленных частот: LSPD — 100.001 - 100.099 EMS — 100.100 - 100.199 LSCSD — 100.200 - 100.299 SANG — 100.300 - 100.399 Government — 100.400 - 100.499 Weazel News — 100.500 - 100.599 FIB — 100.600 - 100.699 Общая частота для всех государственных организаций — 100.000
4Общение по волнам государства
Общение с использованием волн государственного назначения должно происходить строго в соответствии с кодексом Этики и Служебного поведения.
5Ситуационные коды и тен-коды
Использование системы оповещения о чрезвычайных и потенциально опасных ситуаций должно происходить строго в соответствии с сводом “ситуационных кодов и тен-кодов”, которые закреплены настоящим закономИсключение: в случае, если нужно добавить важное уточнение о ситуации, а соответствующего “тен-кода” не закреплено в настоящем законе.
5.1Тен-код система сокращения радио
“Тен-код” — система кодов, используемая для стандартизации и сокращения радиопередач между сотрудниками государственных организаций. Каждый код представляет собой число, как правило начинающееся с 10, за которым следует конкретное значение.
5.2Тен-коды радиопередач
Реестр тен-кодов, используемых при радиопередачах на территории штата San-Andreas: Тен-коды 10-0 — Отмена последнего сообщения 10-3 — Радиомолчание 10-4 — Принял сообщение/информацию 10-5 — Повторить сообщение 10-6 — Не принял сообщение 10-8 — Заступил на патруль 10-9 — Состояние критическое 10-10 — Закончил патрулирование 10-20 — Сообщите свое местоположение 10-21 — Провожу задержание/арест 10-22 — Выехал на место инцидента 10-32 — Зрительный контакт с вооруженными лицами 10-35 — Запрос на посадку/взлет 10-36 — Запрос на движение в воздушном пространстве организации 10-38 — Запрос на посадку/взлет принят 10-39 — Запрос на посадку/взлет отклонен 10-46 — Произвожу обыск 10-48 — Требуется эвакуация 10-49 — Требуется медицинская помощь 10-50 — Веду преследование 10-55 — Произвожу траффик-стоп 10-66 — Запрос подкрепления при 10-50 10-70 — Требуется подкрепление 10-77 — Время прибытия 10-99 — Ситуация урегулирована MAYDAY — совершаю экстренную посадку Ситуационные коды CODE 0 — сотрудник ранен/убит CODE 1 — был открыт огонь CODE 2 — поддержка без “сирен” CODE 3 — поддержка “с сиренами” (приоритетная ситуация) CODE 4 — ситуация стабилизирована CODE 5 — ложное сообщение об инциденте CODE 6 — массовые беспорядки CODE 7 — скопление вооруженных лиц CODE 8 — попытка похищения человека CODE 9 — Требуется переговорщик на место инцидента CODE BROWN — захват/теракт здания/территории CODE BLACK — похищение/покушение на первое лицо штата
5.3Средства связи госслужащих
Сотрудники государственных организаций, находясь при исполнении своих должностных обязанностей, обязаны иметь при себе все средства связи и рацию, позволяющие принимать сообщения через волны государственного назначения и систему оповещения о чрезвычайных и потенциально опасных ситуациях.Исключение: наличие средства связи, способного принимать сообщения по диапазону частот 100.000 - 100.699 не обязательно, его обязательное наличие во время исполнения должностных обязанностей регламентируется внутренним уставом организации.
5.4Реагирование на вызовы
В случае, если поступила информация, сообщаемая в систему оповещения о чрезвычайных и потенциально опасных ситуациях, сотрудники государственных организаций обязуются реагировать на подобные сообщения и вызовы и прибывать на территорию предполагаемого инцидента.
5.5Ложный вызов
Злонамеренное использование системы оповещения о чрезвычайных и потенциально опасных ситуациях и волн государственного назначения с целью ложного вызова сотрудников государственных структур может быть рассмотрено как злоупотребление.

Глава 5. Приоритетное использование волн государственного назначения

1.1Команда экстренного прерывания
Команда «BREAK, BREAK, BREAK» используется для экстренного прерывания радиоэфира в случае:•угрозы жизни сотрудника или гражданина;•нападения;•активного преследования;•иных неотложных ситуаций.
4.1Режим Emergency Traffic Only
Определение Режим Emergency Traffic Only (далее - ETC) — ограничение на использование государственной радиосвязи, при котором передача сообщений разрешена только по приоритетному инциденту, повлекшему его введение.
4.2Основания введения режима ETC
Основания Режим может быть введён в случае:•теракта или угрозы его совершения;•захвата или нападения на объекты госструктур;•захвата/угрозы жизни первых лиц штата;•введения чрезвычайного или военного положения;•рейдовые мероприятия.
4.3Введение режима ETC
Ввод в действие Режим ETC может ввести:•руководящий состав LSPD, LSCSD, FIB, SANG, USSS/USMS;•Губернатор, Вице-губернатор, Министр Обороны и Министр Национальной Безопасности, а также их заместители;Формула ввода: > «ETC. Радиоканал зарезервирован для координации приоритетного инцидента.» > > Нажмите, чтобы раскрыть...
4.4Порядок использования эфира
Порядок использования эфира•Запрещается любая передача, не связанная с основным инцидентом.•Исключение составляют параллельные критические происшествия, требующие немедленной координации между госструктурами.
4.5Команда BREAK при ETC
Совмещение с командой BREAK Во время ETC допускается команда «BREAK, BREAK, BREAK» для экстренного кратковременного прерывания даже внутри зарезервированного эфира. По завершении — обязательно произносится: > «Эфир свободен. Продолжаем ETC.» > > Нажмите, чтобы раскрыть...
4.6Завершение режима ETC
Завершение режима ETC1.Режим ETC может быть отменён только:•структурой, которая его ввела;•уполномоченными представителями Правительства штата:Губернатором, Вице-Губернатором, Министром Национальной Безопасности и Министром Обороны, а также их заместителями
4.7Формула отмены ETC
Формула отмены: > «ETC. Эфир свободен для общего использования.» > > Нажмите, чтобы раскрыть...
