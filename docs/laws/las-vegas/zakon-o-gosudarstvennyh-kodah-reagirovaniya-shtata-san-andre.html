Раздел 1. Общее положение

1.2Справочный характер закона
Закон носит информационно-справочный характер и направлен на обеспечение единообразного понимания и применения служебных сигналов, сообщений и кодов.**
2Государственные коды реагирования
Цели и задачи
2.1Цели госкодов реагирования
Целями настоящего Закона являются:–повышение эффективности межведомственного взаимодействия;–сокращение времени передачи и обработки информации в чрезвычайных ситуациях;–обеспечение точности и однозначности служебных сообщений.
2.2Задачи системы госкодов
Задачами Закона являются:–установление перечня кодов и протоколов, применяемых на территории штата;–определение правил их использования.

Раздел 3. Тен-коды

3.1Применение тен-кодов
–Тен-коды используется для сокращения передачи важной информации.
3.2Приоритет тен-кодов
–В случае возможности использования тен-кода для передачи информации, тен-код должен обладать приоритетом.
3.3Список тен-кодов
–Тен-кодами считаются:> **10-0 — Отмена или отбой; > 10-1 — Не принял сообщение, повторите; > 10-3 — Прекратить трансляцию; > 10-4 — Принято; > 10-5 — Повторите последнее сообщение; > 10-6 — Не принял/нет/отрицательно; > 10-7 — Конец связи; > 10-8 — Готов к работе; > 10-12 — В «FIB» посетитель; > 10-14 — Проникновение в «FIB»; > 10-15 — Возвращаюсь в «FIB»; > 10-20 — Локация или Местоположение; > 10-21 — Направляюсь на локацию; > 10-33 — Требуется эвакуация; > 10-43 — Запрос информации; > 10-46 — Провожу обыск задержанного; > 10-50 — Выехал на поддержку или выезжаю на подкрепление; > 10-70 — Требуется подкрепление/запрос поддержки; > 10-71 — Требуется медицинская помощь; > 10-75 — Провожу задержание; > 10-99 — Ситуация урегулирована или отбой.** > > Нажмите, чтобы раскрыть...

Раздел 4. Ситуативные коды

4.2Ситуативные коды реагирования
–Ситуативными кодами признаются:**> **CODE 0 - Офицер убит или тяжело ранен; > CODE 1 - Офицер ранен; > CODE 2 - Обычный вызов помощи; > CODE 3 - Срочный вызов; > CODE 4 - Не требуется помощь на ситуации или ситуация окончена; > CODE 5 - Ложная тревога; > CODE 6 - Массовые беспорядки; > CODE 7 - Требуется срочная огневая поддержка/массовое скопление криминальных элементов; > CODE 8 - Похищение или попытка похищения человека; > CODE 9 - Угроза охраняемому лицу.** > > Нажмите, чтобы раскрыть...
