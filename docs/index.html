Это живое приложение — попробуй: «мокруха», «дк 39», «ограбление банка»

Понимает игрока

«Ствол без лицензии» найдёт закон об обороте оружия, «мокруха» — статьи об убийстве. Опечатки прощаются, номера статей — святое.

Наказание — сразу

Штраф, срок, звёзды розыска и залог видны прямо в списке результатов. Когда тебя держат на мушке — некогда кликать.

Что изменилось

База сравнивает себя с прошлыми версиями и показывает изменения по статьям: новые составы, правки санкций. Смотри ченджлог.

Оверлей поверх игры

Windows-приложение: глобальный хоткей Ctrl+Shift+L, пиновка статей поверх экрана, работа без интернета. Без инъекций в игру — обычное окно, GTA в режиме «оконный без рамки».
